# Development tips

## Show part as the code is updated

Run in terminal:
```
.venv/bin/python watch.py
```
or
```
source .venv/bin/activate
chmod a+x watch.py
./watch.py
```

Configure IDE to save file immediately. Add to `settings.json`:
```
    "files.autoSave": "afterDelay",
    "files.autoSaveDelay": 500
```
